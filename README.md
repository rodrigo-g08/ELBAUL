# OS_PC4_ElBaul

Backend para ElBaul, e-commerce de productos de segunda mano con funciones sociales.

## Stack Tecnológico

- Node.js 18+
- Express.js
- MongoDB con Mongoose
- JWT para autenticación
- Docker para containerización

## Instalación

### Prerrequisitos
- Node.js 18+
- Docker Desktop
- Postman

# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Producción
npm start
